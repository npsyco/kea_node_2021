Commands

This is an overview of the relevant commands for using node.js.

Description Command Details
Description Command Details
Install dependencies npm i(nstall) This will look through the package.json file and install dependencies defined within this file. This can also utilze installing a package by adding the <package>.
Run script npm run <script name> This will run a named script, where many variables can be utilized (ie. port number).
Start program node app.js This runs the specified .js file in the node server environment.
Install package(s) npm i <package> This installs packages. This command comes in different varieties; mainly whether the package is to be installed for the specific project or as a globally accessible package, that all programs can utilize.
