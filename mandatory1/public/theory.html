Theory

  • Node.js is an open source runtime environment for server-side network applications. Node.s is single-threaded and uses Goggles V8 Engine to execute code. It is cross-platform and runs on Win, Linux and more.
  • It provides event-driven architecture. The event loop allows for seamless data update.
  • Node.js has an async nature (Asynchronous I/O), permitting the processing of other processes before a transmission has finished.
